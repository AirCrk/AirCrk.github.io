解决Kali Linux开机黑屏只显示鼠标

今天在图形化界面下手动删除了一个新建的账户，重启后无法正常进入系统，一直卡在鼠标（钟表）图标上，但鼠标可以动，谷歌了一番后得到了解决，只需要重新装下X-Window即可。 需要的几条命令 apt-get update apt-get install x-window-system apt-get install xorg startx
